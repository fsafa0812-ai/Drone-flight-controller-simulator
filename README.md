# 🚁 Drone Flight Controller Simulator

A real-time drone flight simulation built using Python and Pygame that demonstrates PID control, physics simulation, wind disturbance, and autonomous landing behavior.

---

## 📌 Project Overview

This project simulates a drone flight system inspired by real-world control systems.

It was built as a learning project to understand:
- How drones maintain stability using PID controllers
- How external forces like wind affect motion
- How real-time physics simulations work
- How autonomous landing systems behave

What started as a simple idea during free time turned into a complete engineering simulation project.

---

## ✨ Features

### ✈️ Flight Control
- Manual altitude control using keyboard (↑ / ↓)
- Smooth horizontal movement (← / →)
- Real-time physics-based motion

### 🤖 PID Controller
- Maintains target altitude
- Reduces oscillations
- Simulates real drone stabilization system

### 🌬️ Wind Simulation
- Random wind disturbances
- Affects drone stability
- Makes flight behavior more realistic

### ☁️ Environment
- Moving clouds animation
- Ground and landing pad
- Sky-based simulation view

### 🛬 Landing System
- Automatic landing mode (L key)
- Smooth descent to landing pad
- Flight modes:
  - ASCENDING
  - DESCENDING
  - HOVER
  - LANDING
  - LANDED

### 📊 Telemetry Dashboard
- Altitude
- Velocity
- Target altitude
- Battery level
- Flight mode
- Wind speed
